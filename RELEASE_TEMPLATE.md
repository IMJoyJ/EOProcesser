# ガイド

## スタンドアロン版またはフレームワーク依存版の選択

このライブラリは以下の2つの方法で利用できます：

### スタンドアロン版

スタンドアロン版は外部依存関係がなく、単独で動作します。

### フレームワーク依存版

.NET 8.0 Frameworkのインストールが必要です。
